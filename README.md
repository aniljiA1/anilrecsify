#React Mindmap

A dynamic and interactive mind mapping tool built with React. Easily visualize ideas, concepts, or projects in a tree-like structure. Fully responsive and customizable.

#structure
src/
├── components/
│   ├── MindmapCanvas.jsx
│   ├── MindmapNode.jsx
│   └── Sidebar.jsx
├── data/
│   └── mindmap.json
├── store/
│   └── mindmapStore.js
├── utils/
│   └── buildGraph.js
├── App.jsx
└── main.jsx

#Features

Interactive mindmap with expandable/collapsible nodes

Add, edit, and delete nodes

Drag and drop functionality for node repositioning

Zoom in/out for better navigation

Responsive design for desktop and mobile

Light and dark theme support (optional)
